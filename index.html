Computer Science Undergrad Writing Games

I am currently a sophomore at UCI, pursuing a Bachelor’s in Computer Science. I've programmed since 2008, with comprehensive guidance from my brothers.

I've participated in hackathons and game jams, learning the workflow of building a project. I've also done competitive coding. I learned data structures and algorithms, as well as the problem solving skills of finding invariants, different representations of the same problem, and reductions of problems.

Programming languages I am currently using are Python and C#, but I can also use Java, C++, Javascript, and more. I can use the game engines Godot and Unity. Previously, I had used LibGDX, a Java game development framework, alongside Ashley, notable for being an Entity Component System. (I really like ECS/DOD!)

I am also highly interested in Math and Physics, and I am considering a minor in one of the two.

More About Me

I have always made games since I began programming, starting from simple command line games. In highschool, I learned to use LibGDX, a game oriented graphics framework in Java. Most of my projects were experiments with it and Ashley, an Entity Component System by the same developers. With these two, I was able to lead a runner-up hackathon project in 2018.

With that, I have decided to move on to using Godot and Unity. I have also joined UCI's Video Game Development Club. With the club, I helped make Can Balls Meow, a Smash-like Game.

Other facts about me are that:

  • I closely follow and occasionally compete in the Super Smash Bros. competitive scene.
  • I am quite good at using Minecraft Redstone.
  • I can T-Spin Triple in Tetris.
  • I can play the piano.
