Programming

My greatest strength and hobby is definitely programming.

I am completely self taught, with very little formal training. Because of this, I have little sense of scale for what is (and isn't) advanced or impressive. Take the upcoming bragging with a grain of salt.

In my opinion, I have moderate experience with algorithms. (Time Complexity, Master Method, Designing Greedy Algorithms) and data structures (Types, Usages, Associated Time Complexities). I have some experience in software design (MVC, Observer Pattern, State Pattern), and am currently working to improve it. I can work with APIs and frameworks, such as Java's LibGDX and Javacord.

I work in Java for a large majority of projects, using Intellij IDEA. I am working on mastering computer science topics through Java first. I have looked into webdev; this site is written by me, using Bootstrap, JQuery, and AngularJS. I have dabbled with threading and full stack development. Programming languages that I have used are BASIC, Logo, Pascal, Java, Python, Javascript, OCaml, and Haskell. I understand basic usage of Git.
